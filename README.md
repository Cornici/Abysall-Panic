# Abysall-Panic
Gemini a spus  Abysall Panic este un horror subacvatic claustrofobic pentru Campfire 2026, unde supraviețuirea ta depinde de gestionarea bateriei și a propriei panici. Navighează prin beznă folosind sonarul și lanterna, dar ai grijă: pe măsură ce îți pierzi cumpătul, peștii inofensivi se transformă în coșmaruri vizuale sub ochii tăi.
